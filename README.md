
# Forest Runner

Repository of a game inspired by Clear Code tutorial Pygame Tutorial


![Sample](sample.png)


## Run Locally

Clone the project

```bash
  git clone https://https://github.com/isaac-evs/Pygame_Tutorial.git
```

Go to the project directory

```bash
  cd Pygame_Tutorial
```

Install dependencies

```bash
  pip3 install pygame 
```

Run the game

```bash
  python3 main.py
```
